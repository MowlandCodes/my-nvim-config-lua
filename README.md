## üìö Key Bindings

| Shortcut               | Action                               |
|------------------------|--------------------------------------|
| `<leader>ff`           | Fuzzy find files                    |
| `<leader>fb`           | Find buffers                        |
| `<leader>gs`           | Git status                          |
| `<leader>q`            | Close Neovim                        |

Customize these bindings in the `config/vim-settings.lua` file for full control over your workflow.
